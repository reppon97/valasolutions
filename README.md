# Vala Solutions Test Task


- Tech stack:
  * Web App - Django
  * Database - PostgreSQL
  * Containerized via docker-compose
  

### Usage 

simply:

`docker compose up`

##### DB migration commands

`docker compose exec -it python manage.py makemigrations`

`docker compose exec -it python manage.py migrate`
